🐍 Snake Game (Python)
This project is a simple implementation of the classic Snake game, inspired by the version that was popular on old keypad mobile phones. The game is developed using Python and recreates the nostalgic gameplay where the player controls a snake to collect food while avoiding collisions.

Features
Classic snake movement
Food generation
Score tracking
Increasing difficulty as the snake grows

Technologies Used
Python
▶️ How to Run
Download or clone the repository
Run the main file:
python main.py

Description
Control the snake to eat food and grow longer. Avoid hitting the walls or yourself. Try to get the highest score!
